# 🧭 QueryPilot — Autonomous Research Agent

> Autonomously researches any topic and delivers a structured, professional report.
> Built with **LangChain**, **Google Gemini**, and **Streamlit**.

---

## 📸 What is QueryPilot?

**QueryPilot** is an AI-powered research agent that takes any topic as input and:

1. 🌐 Searches the web for recent, relevant information
2. 📚 Queries Wikipedia for foundational knowledge
3. 🧠 Reasons across multiple sources (ReAct loop)
4. 📋 Generates a structured, downloadable research report

## ⚡ Quick Start (5 Steps)

### 1 — Clone or download the project
```bash
git clone https://github.com/yourusername/QueryPilot.git
cd QueryPilot
```

### 2 — Create virtual environment
```bash
python -m venv venv

# Activate
source venv/bin/activate        # Mac / Linux
venv\Scripts\activate           # Windows
```

### 3 — Install dependencies
```bash
pip install -r requirements.txt
```

### 4 — Add API keys
```bash
cp .env.example .env
# Edit .env and fill in your keys
```

`.env` contents:
```env
GOOGLE_API_KEY=your_gemini_key_here
TAVILY_API_KEY=your_tavily_key_here    # optional
```

**Get your free keys:**
- Google Gemini → [aistudio.google.com](https://aistudio.google.com)
- Tavily Search → [tavily.com](https://tavily.com) *(optional — DuckDuckGo used if absent)*

### 5 — Launch QueryPilot
```bash
streamlit run app.py
```

## 💻 Usage

### Streamlit UI
1. Enter a research topic
2. Click **🚀 Launch Research**
3. Watch the live ReAct reasoning feed
4. Read the formatted report
5. Download as `.txt` or `.md`

### Command Line
```bash
python main.py "Impact of AI in Healthcare"
python main.py "Future of Renewable Energy"
```

### Python API
```python
from dotenv import load_dotenv
load_dotenv()

from main import generate_research_report

result = generate_research_report("Quantum Computing", save_to_file=True)
print(result["report_md"])
```

## 🧠 How QueryPilot Works

```
User Input (Topic)
       │
       ▼
  Streamlit UI (app.py)
       │
       ▼
  Research Pipeline (main.py)
       │
       ▼
  ReAct Agent (agent.py)
  ┌─────────────────────────┐
  │ Thought → Action        │
  │ Action  → Tool Call     │
  │ Observe → Next Thought  │
  │ (repeats 5-8 times)     │
  └──────────┬──────────────┘
             │
      ┌──────┴──────┐
      ▼             ▼
  Web Search    Wikipedia
  (tools.py)    (tools.py)
             │
             ▼
  Report Formatter (report_generator.py)
             │
             ▼
  📋 Structured Markdown Report
```

```
  ...

## Future Scope

  ...

## Conclusion
  2-3 paragraph summary
```

## 🔧 Troubleshooting

| Problem | Fix |
|---------|-----|
| `GOOGLE_API_KEY not found` | Create `.env` with your key |
| `ModuleNotFoundError` | Activate venv → `pip install -r requirements.txt` |
| DuckDuckGo rate limit | Add Tavily key, or wait 30s |
| Report takes too long | Normal — research takes 1-3 minutes |
| Windows `activate` error | Run `Set-ExecutionPolicy RemoteSigned` in PowerShell |
